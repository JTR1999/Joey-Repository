This is my version of the CSS Zen Garden. CSS Zen Garden is a webpage example of using CSS to make websites. The trick is that the only changes that are made are to the CSS, not the HTML. I was given the HTML and CSS files with the task of making my own website by only altering the CSS. This was a very early example of my work. There are thousands of things I could have improved, but it is a snapshot of my progress as a programmer. If you wish to see other entries or try the challenge yourself, you may see their official website here.

While I didn't have as much involvment in the coding of the website, I was a much bigger influence in the design. Our project wasn't the most graphically impressive, but had the most interesting ways to present a lot of information.
